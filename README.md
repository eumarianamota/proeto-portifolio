# Portifólio Pessoal
Este é o repositório do meu portfólio pessoal, onde concentro meus principais projetos, experiências acadêmicas e profissionais.

[Acesse o site aqui](https://eumarianamota.vercel.app/)

## Tecnologias utilizadas
- **Framework:** [NextJS](https://nextjs.org/)
- **Estilização:** [Tailwind](https://tailwindcss.com/)
- **Lingauegm:** [Typescript](https://www.typescriptlang.org/)
- **Deploy:** [Vercel](https://vercel.com/)

## Como rodar o projeto localmente 
### 1. Clone o repositório 
```
git clone https://github.com/eumarianamota/portfolio.git
```

### 2. Instale as dependências
```
npm install
```

### 3. Inicie o servidor de desenvolvimento 
```
npm run dev 
```
